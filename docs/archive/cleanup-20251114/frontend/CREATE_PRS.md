## How to Create PRs

1. Go to: https://github.com/najuna-brian/medipact/pulls
2. Click "New Pull Request"
3. Select the branch from the dropdown
4. Copy the description from PR_DESCRIPTIONS.md
5. Click "Create Pull Request"

## Check Which PRs Already Exist

You can check existing PRs at: https://github.com/najuna-brian/medipact/pulls

Then create PRs for the branches that don't have PRs yet.
